# MST-E Dataset Integration

## Role

MST-E supports perceived skin-tone annotation analysis, ordinal-tone robustness evaluation, and calibration examples.

It is not treated as absolute colorimetric truth.

## Governance State

The original source was preserved without modification. Corrections are applied through governed overlays.

Evaluation must use subject-disjoint folds or leave-one-subject-out analysis to prevent identity leakage.

## Public Repository Scope

Permitted artifacts include aggregate readiness summaries, annotation schemas, correction methodology, conflict classifications, and evaluation protocols.

Media, canonical record tables, subject identifiers, and subject-to-fold assignments remain outside Git.
