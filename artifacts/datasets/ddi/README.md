# DDI Dataset Integration

## Role

DDI is used as an external dermatology evaluation set for robustness, diagnostic performance, and skin-tone-related subgroup analysis.

It is not treated as instrument-calibrated skin-color ground truth.

## Validated Local State

- 656 images
- package completeness validated locally

## Public Repository Scope

Only aggregate integrity results, label schema summaries, evaluation protocols, and aggregate performance results may be published.

Images, image-level metadata, diagnoses paired with filenames, and source manifests remain outside Git.
