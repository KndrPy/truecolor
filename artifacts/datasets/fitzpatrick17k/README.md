# Fitzpatrick17k Dataset Integration

## Role

Fitzpatrick17k is used as a dermatology fairness and robustness benchmark.

Its Fitzpatrick labels are ordinal phenotype annotations and are not equivalent to calibrated CIELAB, ITA, melanin index, or spectrophotometer measurements.

## Public Repository Scope

Permitted artifacts include aggregate package validation, label distributions, schema descriptions, robustness protocols, and aggregate evaluation results.

Images, source metadata tables, URLs, image-level labels, and image-level split assignments remain outside Git.
